PREDICTING THE LOAN DATA INFORMATION

Quick Introduction


The objective of this program is to evaluate the operation of a pipeline on the loan prediction dataset which completes our data modeling operation for classification tasks.

The program implements a session where the (pickle) serialized pipeline is stored in the database and can be further used to make predictions on new information, which should be manually imputed to the form for checking model predictions.



How to test this program


Start by serializing the sklearn pipeline from the activities we did on Data Modelling for classification tasks and uploading it. After uploading the pipeline, head to making a prediction on the prediction page.



The Genesis


The simple application aims to give us a visual of how the sklearn.pipeline.Pipeline can be serialized and deployed for making predictions on a live web server.

The topic started with the DSN-UNIILORIN cohort 2, where we talked about data modeling machine learning for classification and regression tasks. The aspect we tackle here though is the classification task - the loan prediction dataset to be precise. We aim to be able to make a prediction of the loan status using the data point (feature columns)

The course lecture has been archived on the official google-drive for the cohort and the material is available in the repository for the DSN UNIILORIN.
